blog制作之旅

下载git：

  1. 由于网页一直转不开，所以病急乱投医地开了vpn解决了该问题
  2. 下载速度过慢了 ，上网搜索加快加速的方法，发现下**idm可以多线程下载 **，so quickly

下载nodejs：

1.又是因为下载太慢，发现原因可能来自于国外网站限速，使用了清华大学开源软件镜像站进行下载

== 制作过程的参考链接来自于【【Hexo + Github 搭建自己的专属博客】 - CSDN App】http://t.csdnimg.cn/lvVDP ==

误打误撞下载虚拟机：

由于我看到以上教程需要使用到Linux写代码，于是我满怀好奇地对其进行了搜索

发现他需要在虚拟机上运行

于是我下载了ubuntu，但是进入Ubuntu时他要求我输入卡密

这时我想起了万能的淘宝（据淘宝的教程，我先是下载了一个Windows的虚拟机，后又自己倒腾了一个cenos的虚拟机，最后才装上了Linux）

对镜像网站的探索：

因为npm是国外的，国内下载会卡

于是我使用了npm mirror进行下载（在这里发现了为什么一开始上hexo下载的文件少了）

给我感觉不同的地方就是把npm改成cnpm了

发现证书验证影响vpn的使用：

如图 在下载过程中无法进入该网站

通过关闭证书验证解决了该问题

网站运行阶段的错误：

在网站运行阶段对网站进行修改

发现没有作用

发现要先写CTRL+C关闭网站才可以修改

将网站部署到GitHub上

部署网站时 我们需要用到ssh密钥跟网站进行连接

但是设置ssh时我一直遇到无法找到该文件这样的问题

在使用管理员身份设置ssh

我才终于解决了这个问题
